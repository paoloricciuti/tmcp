# Plan: Add `disableSse` Option to @tmcp/transport-http

## Overview

Add an optional `disableSse` boolean option to `HttpTransportOptions` that, when set to `true`, returns a `405 Method Not Allowed` response with a `null` body for GET requests instead of establishing an SSE stream.

## Files to Modify

1. **`packages/transport-http/src/index.js`** - Main implementation
2. **`packages/transport-http/README.md`** - Documentation
3. **`packages/transport-http/test/index.test.js`** - Tests

```typescript
interface HttpTransportOptions {
    getSessionId: () => string;
    path?: string | null;
    oauth?: OAuth;
    sessionManager?: {...};
    disableSse?: boolean; // Disable SSE stream (GET returns 405)

```

Add a dedicated section explaining the option:

```markdown
### Disabling SSE Streams

If your deployment doesn't support long-lived SSE connections (e.g., some serverless environments), you can disable the GET endpoint:

\`\`\`javascript
const transport = new HttpTransport(server, {
disableSse: true,
});
\`\`\`

When `disableSse` is `true`, GET requests to the MCP endpoint will return a `405 Method Not Allowed` response. Clients can still communicate via POST requests, but won't receive server-initiated notifications.
```

### 6. Add tests (test/index.test.js)

Add a new `describe` block for `disableSse` tests with both unit-style and integration tests:

```javascript
describe('disableSse option', () => {
	// Unit-style test using direct transport.respond()
	it('returns 405 for GET requests when disableSse is true', async () => {
		const transport = new HttpTransport(mcp_server, {
			path: '/mcp',
			disableSse: true,
			getSessionId: () => 'test-session-id',
		});

		const response = await transport.respond(
			new Request('http://localhost/mcp', {
				method: 'GET',
				headers: {
					'mcp-session-id': 'test-session-id',
				},
			}),
		);

		expect(response).not.toBeNull();
		expect(response?.status).toBe(405);
		expect(response?.headers.get('Allow')).toBe('POST, DELETE, OPTIONS');
	});

	it('allows GET requests when disableSse is false or undefined', async () => {
		const transport = new HttpTransport(mcp_server, {
			path: '/mcp',
			disableSse: false,
			getSessionId: () => 'test-session-id',
		});

		const response = await transport.respond(
			new Request('http://localhost/mcp', {
				method: 'GET',
				headers: {
					'mcp-session-id': 'test-session-id',
				},
			}),
		);

		expect(response).not.toBeNull();
		expect(response?.status).toBe(200);
		expect(response?.headers.get('Content-Type')).toBe('text/event-stream');
		await response?.body?.cancel();
	});
});
```
